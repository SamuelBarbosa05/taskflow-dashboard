# TaskFlow Dashboard

Aplicação web estilo Kanban desenvolvida com React, com suporte a drag and drop, gerenciamento de tarefas e interface moderna.

## Funcionalidades

- Criação de tarefas
- Edição e exclusão
- Drag and drop entre colunas
- Sistema de prioridade (alta, média, baixa)
- Busca de tarefas
- Modo escuro
- Persistência com localStorage
- Interface responsiva

## Tecnologias

- React
- Vite
- JavaScript
- Tailwind CSS
- @dnd-kit/core

## Como rodar

```bash
npm install
npm run dev
