# 📋 TaskFlow Dashboard

Modern productivity dashboard built to improve task organization, workflow visibility and user experience.

## 🔗 Live Demo

https://taskflow-dashboard-two.vercel.app/

---

## 📌 Overview

TaskFlow Dashboard is a modern task management platform inspired by productivity SaaS applications like Trello, Notion and Asana.

The project was developed to demonstrate practical front-end engineering skills, focusing on responsive UI, clean component architecture and polished user experience.

Designed with a professional dashboard aesthetic, TaskFlow helps users visualize tasks, manage workflow stages and improve productivity.
